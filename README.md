# Portal Input Nilai Mahasiswa

**Portal Input Nilai Mahasiswa** adalah aplikasi berbasis web yang dirancang untuk mempermudah dosen atau admin akademik dalam mengelola, menginput, dan menganalisis nilai mahasiswa secara efisien dan transparan. Aplikasi ini dilengkapi dengan visualisasi data untuk melacak perkembangan akademik mahasiswa.

---

## 🚀 Fitur Utama

Aplikasi ini dilengkapi dengan berbagai fitur interaktif untuk manajemen nilai yang lebih baik:

1. **Input Data & Nilai Mahasiswa**
   * Formulir input yang dinamis dan valid.
   * Mendukung penginputan data esensial: **Nama, NIM, Kelas, Mata Kuliah, dan Nilai**.

2. **Statistik & Diagram Perkembangan**
   * Visualisasi data interaktif menggunakan grafik/diagram.
   * Memudahkan dosen untuk melihat tren kenaikan atau penurunan performa akademik mahasiswa dari waktu ke waktu.

3. **Dashboard Kelas Unggulan**
   * Halaman utama (*dashboard*) yang menyajikan ringkasan performa akademik.
   * Menampilkan daftar **Kelas Unggulan** yang dihitung secara otomatis berdasarkan rata-rata nilai tertinggi.

4. **Riwayat Nilai Berdasarkan Kelas**
   * Log data yang menampilkan semua nilai yang telah diinputkan.
   * Dilengkapi fitur filter otomatis untuk mengelompokkan data berdasarkan kelas tertentu.

5. **Fitur Tambahan (Lain-lain)**
   * **Autentikasi Pengguna:** Sistem *Login* aman untuk Dosen dan Admin Academic.
   * **Pencarian & Filter Cepat:** Mencari data mahasiswa berdasarkan NIM atau Nama secara *real-time*.
   * **Ekspor Laporan:** Fitur untuk mengunduh riwayat nilai dalam format PDF atau Excel *(Optional/Sesuai pengembangan)*.
